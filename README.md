> 🔗 **Live site → [krishnatejaj.github.io/gcp-guide](https://KrishnaTejaJ.github.io/gcp-guide)**

---

## ✨ Key Features

* **🗂️ Category Sidebar:** Browse 30+ GCP services organized into 7 focused categories — Storage, Processing, Messaging, ML/AI, Security, Ops, and Infrastructure.
* **🔍 Global Search & Tag Filters:** Search across all services instantly, or filter by Real-time, Batch, Serverless, SQL, and NoSQL.
* **🧠 Exam Insights:** Every service includes a highlighted callout of what actually shows up in PDE exam questions.
* **🏗️ Real-World Case Studies:** 10 real company architectures (Revolut, Etsy, NatWest Markets, Macquarie Bank, and more) mapped to the exam concepts they demonstrate.
* **🛠️ Deep Dive Panels:** Click any service to expand a full detail panel — cost model, coding level, availability/SLA, IAM roles, open source equivalents, and common troubleshooting.

## 🚀 Quick Start

### Prerequisites
* Node.js (v14 or higher)
* npm

### Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/KrishnaTejaJ/gcp-guide.git
    cd gcp-guide
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Run locally**
    ```bash
    npm start
    ```
    Open [http://localhost:3000/gcp-guide](http://localhost:3000/gcp-guide) to view it in the browser.
