# GCP Data Engineer Study Guide

A clean, interactive reference for the **Google Cloud Professional Data Engineer** certification exam.

**Live site → [krishnatejaj.github.io/gcp-guide](https://KrishnaTejaJ.github.io/gcp-guide)**

---

## What's inside

**30+ GCP services** organized into 7 categories — Storage & Databases, Processing & Integration, Messaging & Orchestration, ML & AI, Governance & Security, Ops & Monitoring, and Infrastructure.

Each service entry covers:
- What it's best used for
- Key exam insight (the things that actually show up in questions)
- Cost model, coding level, availability / SLA
- Core IAM roles
- Open source equivalent
- Common troubleshooting scenarios
- Link to official docs

**10 real-world case studies** from companies like Revolut, Etsy, NatWest Markets, and Macquarie Bank — each mapped to the PDE exam concepts they demonstrate.

---

## How to use it

- Browse by category using the sidebar (or swipe through tabs on mobile)
- Use **All Tools** to see everything at once
- Filter by tag: Real-time, Batch, Serverless, SQL, NoSQL
- Search across all services from the top bar
- Click any row to expand the full detail panel
