A comprehensive, interactive study companion for the **Google Cloud Professional Data Engineer (PDE) Exam (2025 Edition)**. 

This application condenses official documentation, architectural best practices, and real-world case studies into a searchable, filterable dashboard.

## ✨ Key Features

* **🔍 Global Search & Filtering:** Instantly filter tools by capability (SQL, NoSQL, Serverless) or keywords.
* **🧠 Exam-Critical Context:** Highlights specific "Rule of Thumb" logic tested in the PDE exam (e.g., *"If > 1 week transfer, use Transfer Appliance"*).
* **🏗️ Real-World Architectures:** dedicated section analyzing implementations by Uber, Spotify, Mercedes-Benz, and more.
* **🛠️ Deep Dive Cards:** Expandable technical profiles covering cost models, IAM roles, and common troubleshooting scenarios.

## 🚀 Quick Start

### Prerequisites
* Node.js (v14 or higher)
* npm

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/yourusername/gcp-guide.git](https://github.com/yourusername/gcp-guide.git)
    cd gcp-guide
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the application**
    ```bash
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🛠️ Tech Stack

* **Frontend:** React (Hooks, Context)
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
